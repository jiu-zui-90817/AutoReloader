# YR/MO Hot Reloader (红警2/心灵终结 独立热重载组件)

一个基于纯正 `YRpp` 库开发的独立动态热重载组件。本组件无需绑定任何特定的扩展平台源码，采用外挂 DLL 幽灵线程架构，可在游戏运行中实时监测并向内存注入修改后的 INI 属性。

针对 **心灵终结 (Mental Omega)** 的调试需求进行了深度优化，完美支持各类复杂的视觉动画与特效（AE系统）的实时预览，极大提升 Mod 制作与调试效率。

---

## 🚀 核心特性

- **开箱即用**：提供预编译版本，配合专用启动器即可一键注入。
- **全要素图纸覆盖**：支持游戏内核心实体与特效图纸的饱和式扫描覆写：
  - **作战单位 (TechnoTypes)**：载具、步兵、建筑、战机
  - **武器与战斗系统**：武器 (Weapons)、弹头 (Warheads)、抛射体 (Bullets)
  - **MO 特效调试 (AE系统)**：动画 (Anims)、粒子 (Particles)、粒子系统 (ParticleSystems)、Voxel动画 (VoxelAnims)
- **多阵列轮询**：支持同时监控多个自定义 INI 文件，方便将武器、单位、特效分模块并行调试。
- **动态控制台**：自带开发者控制台日志（支持热插拔显隐控制），格式严谨，方便追踪内存覆写流。
- **优雅的泛型架构**：核心注入引擎采用 C++ 模板化设计，具备极高的执行效率与拓展性。

## 📥 安装与使用 (普通用户)

如果你只想使用本工具来调试你的 Mod，无需配置复杂的编译环境，请直接按照以下步骤操作：

### 1. 获取组件
前往本仓库的 [Releases](../../releases) 页面，下载最新打包的压缩包。将解压出的 `AutoReloader.dll` 和 `ReloaderConfig.ini` 以及 `MOInjector.exe` 放入你的游戏根目录。


### 2. 启动游戏
**必须使用专用启动器** 使用MOInjector.exe启动游戏。组件成功挂载后，会根据配置自动弹出调试终端。

## 🛠️ 编译指引 (开发者)

如果你想基于本组件进行二次开发，请参考以下配置：

- **IDE**: Visual Studio 2019 / 2022 (工具集 v142 / v143)
- **依赖库**: 需要自行配置好 `YRpp` (尤里复仇 C++ 扩展库) 以及相关环境。
- **编译建议**: 编译时建议在项目属性中关闭“符合模式 (Conformance Mode: No)”，并使用 C++17 标准，以获取最佳的 YRpp 兼容性。编译出的 DLL 文件按上述普通用户流程挂载即可。

---

## ⚙️ 配置文件说明 (`ReloaderConfig.ini`)

在游戏根目录下修改 `ReloaderConfig.ini`，可实时控制重载器的行为（修改后无需重启游戏）：

```ini
[Settings]
; 自动监控开关。true: 保存目标 INI 文件即刻在游戏内生效；false: 需手动按热键触发
AutoMonitor=true

; 手动重载触发热键。默认 116 代表 F5 键 (采用 Windows 虚拟键码)
HotKey=116

; 调试控制台开关。true: 显示终端黑框以查看注入日志；false: 隐藏黑框
ShowConsole=true

; 目标监控文件列表。支持监控多个文件，用英文逗号分隔即可。这些文件需放在游戏根目录。
TargetINI=hotfix.ini, test_weapons.ini, ae_effects.ini
